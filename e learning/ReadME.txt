# E-Learning Website

## 📚 Project Overview

E-Learning is a responsive educational website developed using HTML, CSS, and JavaScript. The project provides a modern online learning platform where students can browse courses, enroll in programs, watch video lectures, and manage their profiles.

The website is designed with attractive UI elements, responsive layouts, modern gradients, interactive components, and professional styling.

---

# 🚀 Features

- Responsive Website Design
- Modern Navigation Bar
- Course Enrollment Form
- Video Lecture Page
- Student Profile Page
- Login Page
- Course Cards
- Interactive Buttons
- Form Validation
- Responsive Grid Layout
- Modern UI/UX Design

---

# 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript
